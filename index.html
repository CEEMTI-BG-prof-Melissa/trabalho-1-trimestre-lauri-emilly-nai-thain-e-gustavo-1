Biblioteca Escolar Digital

Requisitos Funcionais

  • Permitir cadastro de alunos no sistema
  • Permitir login e autenticação de usuários
  • Disponibilizar acesso a livros digitais
  • Permitir reserva de livros físicos
  • Permitir renovação de empréstimos
  • Exibir histórico de empréstimos do aluno
  • Permitir busca de livros por título, autor ou categoria
  • Permitir solicitação de materiais complementares

Requisitos Não Funcionais

  • O sistema deve ser acessível via navegador web
  • Interface responsiva para dispositivos móveis
  • Tempo de resposta inferior a 3 segundos
  • Garantir segurança dos dados dos usuários
  • Disponibilidade mínima de 99% do sistema
  • Compatibilidade com principais navegadores
  • Interface intuitiva e fácil de usar
  • Suporte a múltiplos usuários simultâneos

Requisitos Inversos

  • O sistema não permitirá acesso sem login
  • O sistema não armazenará senhas em texto puro
  • O sistema não permitirá reservas duplicadas do mesmo livro
  • O sistema não permitirá empréstimos com pendências
  • O sistema não exibirá dados pessoais de outros usuários
